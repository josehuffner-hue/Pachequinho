# 🚀 José Pacheco - Tech Blog

Este é um projeto de um blog de tecnologia minimalista, responsivo e estático, focado em compartilhar artigos sobre desenvolvimento de software, algoritmos e engenharia de software.

O primeiro artigo do blog aborda o funcionamento de **Estruturas de Dados Dinâmicas (Listas Encadeadas)** comparadas a Arrays tradicionais.

ℹ️ **Status do Projeto:** Concluído / Funcional

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web padrão (Vanilla Web), sem a necessidade de frameworks complexos, garantindo leveza e performance:

* **HTML5** - Estruturação semântica do conteúdo do blog.
* **CSS3** - Estilização customizada com paleta de cores moderna (Dark Mode) e variáveis CSS.
* **JavaScript (ES6)** - Manipulação leve de DOM para contagem interativa de curtidas.

---
